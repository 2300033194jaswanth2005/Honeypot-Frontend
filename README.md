# AI-Enhanced Honeypot - Frontend

React.js dashboard for real-time honeypot monitoring and threat intelligence visualization.

## Features
- Real-time attack monitoring
- Interactive charts and graphs
- Attacker profile cards with threat scores
- Dynamic service creation interface
- Export functionality (CSV/JSON)

## Installation
```bash
npm install
npm start
```

Opens at http://localhost:3000

## Tech Stack
React.js, Socket.IO Client, Recharts, Axios
